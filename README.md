# Разработка финансового виджета

## Краткое описание 
Проект является частью учебного процесса и предназначен для 
отработки практических навыков по программированию на Python
и работе с инструментарием разработчика (Pycharm, Poetry, GIT и т.п.)
Проект находится в стадии непрерывного развития. 

## Установка и использование
1. Клонируйте репозиторий:
```
git clone https://github.com/vicburnew/Home_work.git
```
2. Установите зависимости:
```
poetry install 
```

## Примеры использования 
На данный момент в проекте реализовано несколько функций, 
расположенных в пакете .src. Вызов функций реализован из 
модуля main.py

7. filter_by_currency()

*функция принимает на вход список словарей, представляющих транзакции. 
    Функция возвращает итератор, который поочередно выдает транзакции,
    где валюта операции соответствует заданной (например, USD).*

* пример входной информации см. фикстуру "transactions" в файле "conftest.py"
* пример выходной информации - см.фикстуру "transactions_USD_1" в файле "conftest.py"

8. transaction_descriptions()

*Генератор, который принимает список словарей с транзакциями и
    возвращает описание каждой операции по очереди*

* пример входной информации см. фикстуру "transactions" в файле "conftest.py"
* пример выходной информации - см.фикстуру "descriptions" в файле "conftest.py"

9. card_number_generator

*Генератор, который выдает номера банковских карт в формате XXXX XXXX XXXX XXXX, где X  —
    цифра номера карты. Генератор может сгенерировать номера карт в заданном
    диапазоне от 0000 0000 0000 0001 до 9999 9999 9999 9999*

*Пример выхода от 1 до 5:*

*0000 0000 0000 0001  
0000 0000 0000 0002  
    0000 0000 0000 0003  
    0000 0000 0000 0004  
    0000 0000 0000 0005*
